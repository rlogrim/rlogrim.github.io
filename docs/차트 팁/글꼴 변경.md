## 필요한 패키지

글꼴을 변경하기 위해서 먼저 ```showtext``` 패키지 설치하고 불러와야 합니다.

```showtext``` 기능을 켜면, 새로 열린 그래픽 장치(graphics devices)는 ```showtext```를 사용하여 텍스트를 출력합니다.

```R
install.packages("showtext") # 패키지 설치
library(showtext) # 패키지 불러오기
```

## 컴퓨터에 설치된 글꼴 사용하기

글꼴 파일이 위치한 경로는 컴퓨터에서 '제어판 > 모양 및 개인 설정 > 글꼴'로 이동해서 설치하려는 글꼴을 검색하고, 글꼴에서 마우스 오른쪽 클릭하여 속성을 클릭하면 확인할 수 있습니다.

```font_add``` 함수 이용해서 컴퓨터에 설치된 글꼴을 R에서 사용할 수 있도록 불러옵니다.

```R
font_add("kopub", "C:/Users/.../AppData/Local/Microsoft/Windows/Fonts/Kopubdotummedium.ttf")
```

## showtext 기능 자동 사용 설정

```showtext``` 기능이 자동으로 켜지도록 설정하고, dpi를 300으로 설정해 줍니다.

기본 dpi가 96으로 설정되어 있어서 보고서용 그래프를 출력하기에는 적합하지 않습니다.

dpi를 300으로 변경하는 것을 추천합니다.

```R
showtext_auto() # showtext 기능 자동 사용
showtext_opts(dpi=300) # dpi 300 설정
```
