## ggplot2 글꼴 크기 단위는?

Complete themes에는 base_size라는 인자가 있음

base_size 단위는 pt이므로 글꼴 크기를 11pt로 설정하고 싶다면 아래처럼 코드를 작성하면 됨

```R
theme_classic(base_size=11)
```

하지만 Complete themes의 base_size 인자로는 geom_text() 등으로 출력되는 글꼴 설정은 바꿀 수 없음

geom_text()의 size 인자 단위는 어떻게 되는지 보면, 3.88이라고 나옴

```R
GeomLabel$default_aes$size
```

1 : 3.88pt = x : 11pt => x = 11 / 3.88 이므로, 아래처럼 size를 입력하면 글꼴 크기가 11pt로 바뀜

```R
geom_text(size=11/3.88)
```
