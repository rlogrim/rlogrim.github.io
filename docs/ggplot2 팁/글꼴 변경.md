## 필요한 패키지

showtext 패키지 설치하고 불러와야 함
showtext 기능이 켜져 있다면, 새로 열린 그래픽 장치(graphics devices)는 showtext를 사용하여 텍스트를 그림

```R
install.packages("showtext") # 패키지 설치
library(showtext) # 패키지 불러오기
```

## 컴퓨터에 설치된 글꼴 사용하기

제어판 > 모양 및 개인 설정 > 글꼴로 이동해서 설치하려는 글꼴을 검색하고 오른쪽 클릭하여 속성에서 경로를 확인할 수 있음

```R
font_add("kopub", "C:/Users/.../AppData/Local/Microsoft/Windows/Fonts/Kopubdotummedium.ttf")
```

## showtext 기능 자동 사용 설정

showtext 기능이 자동으로 사용되도록 하고 dpi를 300으로 설정해줌

```R
showtext_auto() # showtext 기능 자동 사용
showtext_opts(dpi=300) # dpi 300 설정
```

## ggplot에서 글꼴 적용

```R
geom_text(family="kopub")
```
